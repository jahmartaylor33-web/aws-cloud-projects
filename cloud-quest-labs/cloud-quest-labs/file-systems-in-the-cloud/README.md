
## File Systems in the Cloud (Amazon EFS)

**AWS Services Used**
- Amazon EC2
- Amazon Elastic File System (EFS)

**What This Lab Demonstrates**
- Creating a managed shared file system
- Mounting EFS to multiple EC2 instances
- Multi-AZ access to persistent storage

**Key Takeaways**
- Implemented shared storage across instances
- Validated file persistence and accessibility
- Understood EFS use cases in distributed systems
