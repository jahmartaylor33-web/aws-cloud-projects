
# AWS Cloud Projects

This repository contains hands-on AWS cloud projects demonstrating foundational cloud engineering concepts such as architecture design, networking, security, monitoring, and serverless computing.

The projects were completed using AWS Cloud Quest hands-on labs and select exercises in a personal AWS account to reinforce real-world AWS design patterns and best practices.”

## Projects Included

### 1. Highly Available Web Application
- Designed a multi-AZ web application using EC2, Application Load Balancer, and Auto Scaling.
- Focused on fault tolerance, scalability, and monitoring.

### 2. Serverless API
- Built a serverless REST API using AWS Lambda, API Gateway, and DynamoDB.
- Emphasized event-driven architecture and cost efficiency.

### 3. VPC Networking & Security Architecture
- Designed a custom VPC with public and private subnets.
- Implemented secure networking using route tables, NAT Gateway, and security groups.

## Skills Demonstrated
- AWS core services (EC2, VPC, IAM, S3, CloudWatch)
- High availability and fault tolerance
- Identity and access management (IAM)
- Cloud networking fundamentals
- Monitoring and logging
- Serverless architecture concepts

## Notes
These projects were completed as hands-on learning exercises to demonstrate cloud engineering fundamentals and design decision-making.
